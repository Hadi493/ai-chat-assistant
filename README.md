## 🛠️ Installation

1. **Environment Setup**
   ```bash
   # Create .env file
   cp .env.example .env
   
   # Add your Gemini API key
   GEMINI_API_KEY=your_api_key_here
   ```

2. **Start Development**
   ```bash
   npm run start
   # Server starts at http://localhost:3000
   ```
